## 9) إصلاح Supabase create_public_order

المشكلة الحالية أن frontend يصل إلى Supabase لكن create_public_order لا يرجع رقم تتبع أو يفشل.

افحص:
src/supabase.ts
scripts/production-hardening-gate.mjs
scripts/debug-create-public-order.mjs إن وجد

المطلوب:

* createPublicOrder(payload) يجب أن يرسل:
  supabase.rpc("create_public_order", { p_order_data: payload })

* الدالة في Supabase يجب أن تكون:
  create_public_order(p_order_data jsonb)

* يجب أن ترجع واحدًا من:
  tracking_code
  tracking_number
  id
  أو string مباشر

الأفضل أن ترجع string tracking_code مباشرة.

لو SQL function غير موجودة أو signature مختلفة:
أنشئ SQL patch غير تدميري في:
supabase/sql/DN_PUBLIC_ORDER_RPC_FIX.sql

يجب أن يكون patch:

* idempotent
* لا يحذف بيانات
* لا يسقط جداول
* لا يعطل RLS
* يعطي execute للـ anon فقط على create_public_order و track_order حسب الحاجة
* يتحقق من الحد الأدنى للحقول المطلوبة
* يولد tracking number بنمط:
  DN-YYYY-XXXXX

الدالة يجب أن تقبل payload مرن يحتوي:
sender_name
sender_phone
sender_city
sender_address
receiver_name
receiver_phone
receiver_city
receiver_address
package_type
weight
pieces
service_type
delivery_price
subtotal
base_price
total
total_price
amount
price
currency
source_domain
captcha_token
payment_method
cod_amount
notes
status
status_history
created_at

إذا جدول orders لا يحتوي بعض الأعمدة، لا تفشل الدالة بسببها؛ استخدم الأعمدة الموجودة فقط أو خزن البيانات الإضافية في metadata/jsonb إن وجد.

يجب أن يعمل test:

$env:DN_RUN_ORDER_TEST="1"
pnpm run production:gate

ويظهر:
PASS: create_public_order RPC succeeded
PASS: Tracking code returned
PASS: track_order RPC succeeded

## 14) Security Headers

في vercel.json أضف أو راجع:

X-Content-Type-Options: nosniff
X-Frame-Options: DENY
Referrer-Policy: strict-origin-when-cross-origin
Permissions-Policy: camera=(), microphone=(), geolocation=()
Strict-Transport-Security: max-age=31536000; includeSubDomains

Content-Security-Policy يجب أن يسمح فقط بما يحتاجه الموقع:

* self
* Supabase URL
* Supabase websocket
* Cloudflare Turnstile
* QR server إن كان مستخدمًا
* الصور https/data/blob
* fonts self/data

لا تجعل CSP يكسر الموقع.
