# DAY NIGHT Admin Auth v3 Assets

The image files in this folder are installed automatically by `scripts/install-local-assets.mjs` during `npm run dev`, `npm run build`, and `npm run build:spa`.

Required local paths:

- `intro-gateway.png`
- `login-reference.png`
- `loading-command-center.png`
- `logo-glass.png`
- `admin-dashboard-reference.png`
- `khalifa-card-reference.png`
- `admin-map.png`
- `login-extra-reference.png`

PNG binaries are downloaded during the asset-install step.
