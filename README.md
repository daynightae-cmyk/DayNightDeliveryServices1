Formula:

```txt
billable_weight = ceil(weight_kg)
subtotal = 95 + ((billable_weight - 1) * 45)
vat = subtotal * 0.05
total = subtotal + vat
```

Create `.env` locally.


## Important Security Rule

Never commit:

```txt
service_role key
sb_secret key
database password
JWT secret
connection string
real backend secrets
```

The frontend must never contain hardcoded production secrets.

# Request Delivery Flow

```txt
Customer Opens /request
        │
        ▼
Loads Cities From Supabase
        │
        ▼
Customer Fills Sender + Receiver Details
        │
        ▼
Price Calculated via calculate_delivery_price RPC
        │
        ▼
Customer Confirms Order
        │
        ▼
Order Submitted via create_public_order RPC
        │
        ▼
Supabase Generates Tracking Number
        │
        ▼
Success Screen Shows Tracking Number
        │
        ▼
Customer Can Track or Open WhatsApp
```

# Tracking Flow

```txt
Customer Opens /tracking?code=DN-2026-XXXXX
        │
        ▼
Frontend Reads Tracking Code
        │
        ▼
Calls track_order RPC
        │
        ▼
Supabase Returns Safe Public Tracking Data
        │
        ▼
Timeline Displays Status
        │
        ▼
Invalid Codes Show Clean Error + WhatsApp Support
```

## Public Tracking Must Not Show

* Sender full phone.
* Receiver full phone.
* Full pickup address.
* Full delivery address.
* COD sensitive data.
* Internal admin notes.
* Driver private details.
* Wallet/payment internals.
